# 🛒 Green Store (Work in Progress)

Green Store is a modern eCommerce web application that I am currently developing.
The goal of this project is to build a clean and user-friendly platform for browsing and purchasing daily products like groceries, food items, and essentials.

> 🚧 This project is still under development. Features and UI are not finalized.

---

## 🧠 Current Status

* Basic UI layout created
* Product data integration (static for now)
* Component-based structure implemented
* Navigation setup using React Router
* Responsive design (partially completed)

---

## 🚀 Features (So Far)

* 🛍️ Product listing (grid/card layout)
* 📦 Product details section
* ⭐ Product ratings display
* 🧾 Basic UI for shopping experience
* 🔄 Reusable components
* 📱 Responsive UI (in progress)

---

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* JavaScript (ES6+)
* React Router

---


## ⚙️ Getting Started

Clone the repository:

```bash id="gs002"
git clone https://github.com/nayan-mia-09/green_store.git
cd green_store
```

Install dependencies:

```bash id="gs003"
npm install
```

Run the project:

```bash id="gs004"
npm run dev
```
